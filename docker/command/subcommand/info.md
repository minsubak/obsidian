## 도커 명령어 - info

- 개요
```txt
`info` 명령어는 도커 시스템에 관한 정보를 출력한다. 도커 데몬이 실행 중인 호스트의 여러 상태 및 구성 정보를 확인할 수 있다.
```

- 명령어 기본 구성
```bash
$docker info
```

- 출력 정보
```bash
Docker Version
Runtime Info (ex, Docker Runtime, Container Count, Image Count, etc)
Storage Driver and Capicity
Network Driver Info
System Resource(CPU, Memory, etc) Info
Security Option
```
