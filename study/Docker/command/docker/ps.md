## 도커 명령어 - ps

- 개요
```txt
`ps` 명령어는 현재 실행 중인 도커 컨테이너의 목록을 표시한다. 실행 중인 컨테이너의 기본 정보를 확인하기 쉽다.
```

- 명령어 기본 구성
```bash
$docker ps -option
```

- 명령어 구성 설명
```bash
-option : 명령에 추가할 옵션
```

- option 태그
```bash
-a, --all   : 중지된 컨테이너를 포함한 모든 컨테이너를 출력
-q, --quiet : 컨테이너의 ID만 표기
-n, --last  : 최근 생성된 컨테이너의 수를 지정
-s, --size  : 각 컨테이너의 사용 중인 디스크 공간을 출력
```
